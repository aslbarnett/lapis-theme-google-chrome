- Download Zip from this repo.
- Extract Zip.
- Open Google Chrome.
- Navigate to `chrome://extensions`.
- Click the toggle and enable **Developer mode**.
- Select **Load Unpacked**.
- Select the **lapis-google-chrome** folder to install.
